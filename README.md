# NetSim

**Netzwerk Simulator für Windows — moderner als Filius, kein Java nötig.**

NetSim ist ein kostenloser Netzwerk-Simulator für Windows, entwickelt speziell für Schüler und Auszubildende. Kein Installer, kein Java — einfach Setup starten und loslegen.

---

## Features

- 🖱️ **Drag & Drop Topologie** — Geräte platzieren und per Kabel verbinden
- ▶️ **Entwurf & Simulationsmodus** — Wechsle per Klick, sieh Pakete animiert fließen
- 🌐 **Webserver, DNS, DHCP & mehr** — Reale Protokolle, simuliert
- 💻 **Terminal mit ping, traceroute, ipconfig** — Auf jedem Gerät verfügbar
- 💾 **Projekte speichern & laden** — Als Datei exportieren, jederzeit wieder öffnen
- 🎓 **Gemacht für die Schule** — Klar, übersichtlich, ohne Einrichtung nutzbar
